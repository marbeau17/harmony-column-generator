## 1. 由起子さんの文体DNA分析結果

### 1-1. 文法・文体分析（日本語専門家A）

| 分析項目 | 実測値・特徴 |
|---|---|
| **文末表現の比率** | 「〜です」30% / 「〜ます」25% / 体言止め 20% / 「〜ですね」5% / 「〜かもしれません」8% / 「〜のです」7% / 句点省略（余韻）5% |
| **一文の平均文字数** | 25〜35文字（極めて短い。60文字以上の文は稀） |
| **接続詞パターン** | 「けれど」（「しかし」より圧倒的に多い）/ 「でも」/ 「そして」/ 「だからこそ」/ 「つまり」は稀 |
| **改行パターン** | 1〜2文ごとに改行。重要フレーズは1行に独立させる。空行を多用し「余白」を作る |
| **漢字・ひらがな比率** | ひらがな多め（約65%）。難読漢字を避け、やさしい印象を維持 |
| **句読点の特徴** | 文末の句点「。」を意図的に省略する場合がある（余韻効果） |
| **ダッシュの使用** | 「——」（全角ダッシュ2つ）を感情の転換・余韻に頻用 |

#### 文末表現の実データ

```
由起子パターン（実例）:
- 「〜ものの中にあります。」（柔らかい断定）
- 「〜かもしれません。」（断定回避）
- 「〜と言われています」（伝聞）
- 「〜ていきます。」「〜ていくのです。」（変化・プロセス）
- 「〜ではないでしょうか。」（読者への問いかけ）
- 「〜してみてください。」（やさしい提案）
- 「〜してみてくださいね」（句点なし+ね、で親密さ）
- 「〜かもしれません。」（可能性の提示）
- 「〜のようなもの。」（比喩での体言止め）
- 「〜のです。」（説明・納得）
```

### 1-3. 構成・ストーリーテリング分析（シナリオライター）

#### 記事の基本構造パターン

```
パターンA「問いかけ型」（最頻出）:
1. 読者への共感的問いかけ（「〜ありませんか？」「〜ではないでしょうか？」）
2. テーマの提示（短い1行フレーズ）
3. 具体的な展開（例え話・たとえば〜）
4. スピリチュアルな視点からの解説
5. 読者へのやさしい行動提案
6. 余韻を残す結び

パターンB「体験・エピソード型」:
1. 季節や日常の描写から入る
2. 個人的な体験やカウンセリングでの気づき
3. その体験が示す普遍的な真理
4. 読者への転用（「あなたも〜」）
5. 希望と勇気を与える結び

パターンC「概念解説型」:
1. 一般的な認識の提示（「〜というと、〜と思われがちです」）
2. 視点の転換（「でも実は〜」「けれど〜」）
3. 深い意味の解説
4. 具体例で理解を助ける
5. 読者の日常に落とし込む結び
```

#### 語りかけの距離感

- **一人称**: 「私」（カウンセリング経験を語るとき）、「私たち」（読者と一緒の立場）
- **二人称**: 「あなた」を多用。「皆さん」は少ない。一対一で語りかける親密さ
- **読者への姿勢**: 「上から教える」ではなく「隣に座って一緒に考える」

### 1-4. 文学的技法分析（小説家）

#### リズム感

由起子さんの文章は「短・短・長」のリズムが特徴的:

```
（実例分析）
「遠回りしてもいい。」      ← 短（10文字）
「立ち止まってもいい。」    ← 短（11文字）
「何度でもルートを変えていい。」← 中（15文字）
「大切なのは、「どう生きたいか」というあなたの意志。」← 長（26文字）
```

#### 余白・間の技法

- 重要な概念を「1行に独立」させる（太字の代わり）
- 「——」（全角ダッシュ2連）で感情の転換を表現
- 文末の句点省略で余韻を残す
- 「……」「…。」で余韻・想像の余地を作る

#### 比喩パターン

| パターン | 実例 |
|---|---|
| **自然比喩** | 「まるで愛が波紋のように広がって」「風の匂い、鳥の声、肌に触れる光」 |
| **旅の比喩** | 「人生はひとつの旅」「東京から京都に行くルート」 |
| **光と闇** | 「心の中に宿る光」「やさしい灯り」「夜の闇の中でさえも」 |
| **成長比喩** | 「芽はあり」「育まれていきます」「根っこにある」 |

### 1-5. ブログ特有の文体分析（ブログマスター）

#### タイトルのパターン

```
1. 名詞止め + 副題: 「霊体験と愛の絆」
2. 問いかけ型: 「嫌なことが人生を豊かにする？」
3. メタファー型: 「人生はひとつの旅。魂が選ぶルートで、自分らしい物語を描いていく」
4. 格言風: 「人の心は、心理学の尺度だけでは計れない。」
5. 提案型: 「プチワーク：あなたが"本当に欲しいもの"は何ですか？」
```

#### 導入文の書き出しパターン

```
1. 共感的問いかけ: 「ふとした瞬間に浮かぶ、「こんなふうになりたい」」
2. 普遍的な事実: 「私たちの暮らしの中で、ときに忘れがちなものがあります。」
3. 季節・日常: 「4月も半ばを過ぎ、新たな環境での生活が始まった方も多いのではないでしょうか。」
4. 引用風: 「「私って、どんな性格なんだろう？」」
5. 宣言: 「私たちはこの世界に生まれる前、「どんな人生を歩みたいか」を魂レベルで決めてきている」
```

#### 読者との距離感

- 「ですね」「ですよね」は使うが、頻度は控えめ（馴れ馴れしくならない）
- 「〜してみてくださいね」の「ね」は記事に1〜2回のみ
- 「！」は使用するが控えめ。「！！」の連続は避ける
- 絵文字は使わない（アメブロ本文中）
- 顔文字も使わない

### 1-6. 結びの特徴

```
由起子さんの記事結びパターン:
1. 行動提案 + 祈り: 「ふと空を見上げてみてください。お天道様は、あなたの中に、今も確かに在ります。」
2. 肯定 + エール: 「今この瞬間、あなたはもう十分にがんばっています。どうか、あなた自身を解放してあげてくださいね。」
3. 祈り・願い型: 「この小さなワークが、あなたの心にやさしい灯りをともしますように」
4. 気づきの提示: 「心のカタチは、人それぞれ。だからこそ、自分の感覚を大切にしてあげてくださいね」
```

## 2. 文体ルール集（AIプロンプト組み込み用 30項目）

### 基本文体ルール（R01〜R10）

| ID | ルール | 重要度 |
|---|---|---|
| R01 | **一文は平均25〜35文字**。最長でも50文字以内。60文字超の文は絶対に書かない | 必須 |
| R02 | **1〜2文ごとに改行**する。3文以上を同じ段落に詰め込まない | 必須 |
| R03 | **重要なフレーズは1行に独立**させる（前後に空行） | 必須 |
| R04 | 「です・ます」調を基本とするが、**体言止めを約20%混ぜる**ことでリズムを作る | 必須 |
| R05 | **「けれど」を「しかし」「だが」より優先**して使う。「しかし」は記事全体で0〜1回まで | 必須 |
| R06 | **文末の句点を意図的に省略**する箇所を作る（余韻効果。記事全体の5〜10%） | 必須 |
| R07 | **「——」（全角ダッシュ2連）**を感情の転換・余韻に使う（記事あたり1〜3回） | 推奨 |
| R08 | **「…」「……」**を控えめに使用（記事あたり2〜4回。AI的な多用は厳禁） | 推奨 |
| R09 | 「短・短・長」のリズムパターンを意識する。短文2つの後に少し長い文を置く | 推奨 |
| R10 | **ひらがなを多用**し、漢字密度を下げる。「一つ」→「ひとつ」、「時に」→「ときに」等 | 必須 |

### 語彙・表現ルール（R11〜R20）

| ID | ルール | 重要度 |
|---|---|---|
| R11 | **断定を柔らかくする**: 「〜です」より「〜かもしれません」「〜と言われています」「〜ではないでしょうか」を多用 | 必須 |
| R12 | **読者への問いかけ**を記事冒頭と中盤に入れる: 「〜ありませんか？」「〜ではないでしょうか？」 | 必須 |
| R13 | **「たとえば」で具体例**を必ず入れる。抽象論だけで終わらせない | 必須 |
| R14 | **「そっと」「ゆっくり」「少しずつ」**等のやさしい副詞を散りばめる（記事あたり各1〜2回） | 必須 |
| R15 | **自然のメタファー**を1記事に2〜3個含める（風、光、水、空、花、木、波紋など） | 推奨 |
| R16 | **「あなた」で読者に語りかける**。「皆さん」「読者の方」は使わない | 必須 |
| R17 | **「私たち」で同じ立場に立つ**表現を含める。上から教える構図にしない | 必須 |
| R18 | **「〜してみてください」「〜してみてくださいね」**でやさしく行動提案する（命令形は厳禁） | 必須 |
| R19 | スピリチュアル用語は**日常語に言い換え**てから使う: 「波動」→「波長（波動）」のように括弧で補足 | 推奨 |
| R20 | **カウンセリング経験に基づく表現**: 「カウンセリングの中で感じるのは〜」「多くの方と向き合う中で〜」を記事に1回は入れる | 推奨 |

### 構成・トーンルール（R21〜R30）

| ID | ルール | 重要度 |
|---|---|---|
| R21 | **導入文は共感から始める**: 読者の悩みや日常の気持ちに「〜ではないでしょうか」「〜ありませんか」で寄り添う | 必須 |
| R22 | **記事の結びは「希望」か「肯定」**で終わる。不安で終わらせない | 必須 |
| R23 | **結びに「〜ますように」「〜してあげてくださいね」**のような祈り・エールを含める | 推奨 |
| R24 | **「でも」「けれど」で視点を転換**するパターンを記事に2〜3回使う: 一般論→由起子視点 | 必須 |
| R25 | **箇条書きは最小限**に。由起子さんは地の文で語る。箇条書きは「具体例の列挙」時のみ | 推奨 |
| R26 | **見出しは「やさしいトーン」**で: 「〜の方法」ではなく「〜してみませんか」「〜ということ」 | 推奨 |
| R27 | **「〜のようなもの」「〜のようなイメージ」**で比喩を柔らかく提示する | 推奨 |
| R28 | **「大丈夫」「それでいい」**等の肯定語を記事後半に配置する | 推奨 |
| R29 | **感嘆符「！」は記事本文全体で2〜3回**まで。多用しない | 推奨 |
| R30 | **「〜しましょう」より「〜してみてください」**を使う。「〜しましょう」は押し付けがましく聞こえる | 必須 |

## 3. 禁止表現リスト

### 3-1. AI臭い表現（絶対禁止）

| カテゴリ | 禁止表現 | 理由 |
|---|---|---|
| **過剰な接続詞** | 「まず」「次に」「さらに」「加えて」「また」の連続 | 箇条書き的で温かみがない |
| **論文調** | 「〜において」「〜に関して」「〜の観点から」「〜を踏まえ」 | 由起子さんは使わない。硬すぎる |
| **定型結び** | 「いかがでしたでしょうか？」「この記事では〜を紹介しました」「参考になれば幸いです」 | 典型的AI/SEO記事の結び。由起子さんの語りとは異質 |
| **まとめ宣言** | 「まとめ」「おわりに」「最後に」のような見出し | 由起子さんは自然に結ぶ |
| **過剰な装飾** | 「非常に」「極めて」「圧倒的に」「驚くべき」 | 由起子さんはやさしい語彙を使う |
| **カタカナ乱用** | 「アプローチ」「メソッド」「ソリューション」「フレームワーク」「コンテンツ」 | 由起子さんはやまとことばを好む |
| **機械的列挙** | 「ポイント1」「ポイント2」「ポイント3」の均等配分 | 語りのリズムを壊す |
| **読者想定の押し付け** | 「あなたは〜に違いありません」「誰もが〜と思うでしょう」 | 由起子さんは「〜かもしれません」 |
| **過剰な共感ポーズ** | 「とてもよくわかります」「お気持ち、痛いほどわかります」の連発 | 1回なら自然、連発はAI臭い |
| **段落冒頭の定型** | 全段落が「〜について」「〜というのは」で始まる | 変化のなさがAI的 |

### 3-2. 由起子さんが使わない表現

| 禁止表現 | 代替表現（由起子風） |
|---|---|
| 「〜すべきです」「〜しなければなりません」 | 「〜してみてくださいね」「〜するといいかもしれません」 |
| 「断言します」「間違いありません」 | 「〜と感じています」「〜と思うのです」 |
| 「絶対に〜してください」 | 「ぜひ〜してみてください」 |
| 「成功するための5つの方法」的な見出し | 「〜を大切にするということ」「〜のために、できること」 |
| 「ぶっちゃけ」「ヤバい」「マジで」 | （カジュアルすぎる口語は使わない） |
| 「エビデンス」「ファクト」「ロジック」 | 由起子さんは感性・直感寄りの語彙 |
| 「〜を解説します」「〜をご説明します」 | 「〜についてお話ししたいと思います」「〜について考えてみましょう」 |
| 「重要なポイントは3つあります」 | （数を前面に出す構成はしない） |
| 「科学的に証明されている」 | 「〜と言われています」「〜とされています」 |
| 「スピリチュアル界隈では」「スピ業界」 | 「スピリチュアルな観点からみると」「スピリチュアルの世界では」 |

### 3-3. トーン禁止

- 読者を焦らせる表現: 「今すぐ」「手遅れになる前に」「放置すると危険」
- 恐怖を煽る表現: 「このままでは」「大変なことになる」「不幸を招く」
- 比較・競争: 「あの人より」「勝ち組」「成功者は皆〜している」
- 商業的すぎる表現: 「お得」「限定」「今だけ」（CTA部分は除く）

## 4. Few-shotサンプル集

以下をプロンプトに含め、AIにスタイルを学習させる。

### サンプル1: 導入文パターン（共感型）

```
私たちの暮らしの中で、
ときに忘れがちなものがあります。
それは、
目に見えないものを敬う気持ちです。

便利さやスピード、目に見える成果が重視される今の時代。
けれど、本当に大切なものは、目には映らないけれど、
心には確かに感じられるものの中にあります。
```

### サンプル2: 比喩と余韻パターン

```
まるで愛が波紋のように広がって、

お天道様は、晴れた日だけ空にいるわけではありません。
曇りの日も、雨の日も、
夜の闇の中でさえも——
お天道様の「目」は、私たちの内側に宿っていて、
いつでも私たちを見守ってくれています。
```

```

「こんなふうになりたい」

たとえば──

「海外で暮らしてみたい」

### サンプル4: 概念の説明パターン

```
「諦める」という言葉。
一般的にはネガティブなものとして捉えられる言葉かと思います。

でも実は、「諦める」には二つの相反する意味があるんです。
一つはポジティブな「受容」としての諦め、
もう一つはネガティブな「拒否」としての諦めです。

この違いを知ることで、人生の選択肢や心の在り方が大きく変わってきます
```

```
遠回りしてもいい。
立ち止まってもいい。

大切なのは、「どう生きたいか」というあなたの意志。

### サンプル6: 自然メタファーと五感

```
風の匂い、鳥の声、肌に触れる光…

慌ただしい毎日の中でも、

```

### サンプル7: カウンセリング経験からの語り

```
私が日々のカウンセリングの中で強く感じるのは、
自信とは生まれつきの才能や特別な能力によって得られるものではなく、

経験の中で学び、成長し、少しずつ自分なりの知識や判断基準が身についていく。

```

### サンプル8: 体験エピソード型

## 5. 文体スコアリング基準

生成された文章の「由起子らしさ」を0〜100で評価するための基準。

### 5-1. スコアリング項目（各10点満点、合計100点）

| # | 評価項目 | 10点（完璧） | 5点（まあまあ） | 0点（不合格） |
|---|---|---|---|---|
| 1 | **一文の長さ** | 平均30文字以下、最長50文字以内 | 平均40文字、最長60文字 | 平均50文字超、長文が多い |
| 2 | **改行・余白** | 1〜2文ごとに改行、重要フレーズ独立 | 段落はあるが詰め込み気味 | 改行少なく壁のようなテキスト |
| 3 | **文末バリエーション** | 5種類以上の文末（体言止め含む） | 3種類程度 | 「です」「ます」のみの単調な繰り返し |
| 4 | **柔らかさ・断定回避** | 「かもしれません」「と言われています」多用 | 時々柔らかい表現あり | 断定調が多い |
| 5 | **読者への語りかけ** | 「あなた」で問いかけ、共感表現あり | 問いかけはあるが定型的 | 読者不在の解説調 |
| 6 | **メタファー・比喩** | 自然系メタファー2〜3個、独自性あり | メタファー1個 | 比喩なし or 陳腐な比喩 |

| 8 | **AI臭さの排除** | 禁止表現が0個 | 禁止表現1〜2個 | 禁止表現3個以上 |
| 9 | **結びの温かさ** | 祈り・エール・肯定で結ばれている | 結びはあるが定型的 | 「いかがでしたか」的な結び |
| 10 | **全体のトーン一貫性** | 最初から最後まで由起子トーン | 部分的にトーンが崩れる | 別人が書いたような箇所がある |

## 付録A: 10人の専門家による総合コメント

### 日本語専門家A（文法・文体）
> 由起子さんの文章の最大の特徴は「一文の短さ」と「余白の多さ」である。現行プロンプトの「一文60文字以内」は緩すぎる。実測値は平均25〜35文字であり、ここの数値を厳格化するだけで生成文の印象は大きく変わる。

### 日本語専門家B（語彙・表現）
> 「けれど」の使用頻度が由起子さんの文体を決定づけている。「しかし」を使った瞬間にAI臭くなる。また、ひらがな化は由起子さんの「やさしさ」の根幹であり、漢字密度を下げることが最も効果的な改善策のひとつ。

### シナリオライター（構成）
> 由起子さんの記事は「共感→転換→深化→希望」の4段構成。現行プロンプトにはこの流れの指示がない。特に「でも実は〜」「けれど〜」による視点転換が記事の核心であり、ここを明示的に指示すべき。

### 小説家（文学的技法）
> 「短・短・長」のリズムと「——」による余韻が文学的な質を生んでいる。これはプロンプトで「リズムパターン」として明示的に指示可能。体言止めの比率指定も重要。

### ブログマスター
> 由起子さんの導入文は「ふとした瞬間に〜」「〜ではないでしょうか」で始まることが多い。これはブログ読者の離脱を防ぐ効果がある。「いかがでしたでしょうか」的な結びを排除し、由起子流の「祈り結び」に統一すべき。

### SEOマスター
> キーワードの自然な挿入は現行プロンプトの3〜5回の指示で十分。問題はSEO最適化のためにAI臭い構成（「まとめ」見出し、番号付きポイント等）が入りがちな点。由起子さんの語り口とSEOの両立には、見出しをやさしいトーンにしつつキーワードを含める工夫が必要。

### スピリチュアルカウンセラー代理
> 由起子さんは「スピリチュアル用語を日常語に翻訳する」ことを常にしている。「波動」だけでは読者に伝わらないので「波長（波動）」と補足する。また、「〜すれば開運する」のような安易な因果は由起子さんの表現にはない。あくまで「〜かもしれません」「〜と言われています」。

### AI/NLPエンジニア
> Few-shotサンプルの品質と量がスタイル再現の鍵。現行プロンプトにはFew-shotが0個。最低4〜5個の高品質サンプルを含めるだけで、文体再現度は劇的に向上する。また、生成後のスコアリングによる品質ゲートは再生成コストとのトレードオフだが、70点未満を弾くだけでも効果は大きい。

### 編集者
